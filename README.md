# Number Guessing Game in (C++)

A simple number guessing game written in C++.


## Features
- 3 difficulty levels (Easy, Medium, Hard)
- Random number generation
- Hints (higher / lower)
- Maximum of 5 attempts
- Replay option

## How to play
- Choose a difficulty level
- Enter numbers within the given range
- Try to guess the correct number within 5 attempts
